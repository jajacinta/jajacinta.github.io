Egg Colour Calculator

ww = White Shell Gene | WW/Ww = Blue Shell Gene | BB/Bb = Tan/Brown Topcoat (Bb = Green, BB = Olive)

Punnett Square Calculator that will calculate all possible chicken egg colours that offspring could lay based on parental genotypes. Will generate a 2x2, 4x4 or 8x8 Punnett Square, visual egg colour map and probabilities.

How to Use & Genotype Examples (Click to Expand)

Type the gene letter combinations for both parents into the input fields below. Depending on how many traits you want to track, you can enter 2, 4, or 6 letters to map different grid scales:

1. Single Trait Example (Monohybrid Cross 2x2)

Tracks just one gene group (e.g., shell base color Blue Ww x Ww ; Parent 1: Ww & Parent 2: Ww). Generates 4 unique outcomes.

Ww
WWW (Blue)Ww (Blue)
wWw (Blue)ww (White)

2. Two Traits Example (Dihybrid Cross 4x4)

Tracks shell base color and brown topcoat depth (e.g., Green WwBb × WwBb). Generates 16 calculation paths mapping blues, greens, olives, whites, tans, and deep browns.

WBWbwBwb
WBWWBB (Olive)WWBb (Green)WwBB (Olive)WwBb (Green)
wbWwBb (Green)Wwbb (Blue)wwBb (Tan)wwbb (White)

3. Three Traits Modifier Example (Trihybrid Cross 8x8)

Tracks shell base colour (White (ww) or Blue (WW/Ww)), brown topcoat (Brown (BB) or Tan (Bb)) , plus an optional third density modifier gene (CC/Cc) (e.g., WwBbCc × WwBbCc). Generates 64 matrix calculations for ultra-precise shade probabilitys.
